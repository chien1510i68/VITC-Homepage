# News Images Directory

This directory should contain images for news articles.

## Required Images:
- news-1.jpg - Khai giảng khóa Tin học văn phòng
- news-2.jpg - Hội thảo xu hướng công nghệ 
- news-3.jpg - Lễ trao chứng chỉ học viên

## Image Specifications:
- Format: JPG or PNG
- Dimensions: 800x450px (16:9 aspect ratio)
- File size: < 500KB for optimal performance
- Content: Professional photos related to VITC activities

Note: Component includes fallback placeholder design if images are missing.
